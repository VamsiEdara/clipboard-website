### Usage

Clone the repo:

```bash
git clone
```

Install the dependencies:

```bash
npm install
```

Build or watch the CSS file:

```bash
# Build once
npm run build

# Watch for changes
npm run watch
```

This will watch the `src/input.css` file and build it to `css/style.css`, which will be your final CSS file.
